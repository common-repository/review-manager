== Installation ==

This section describes how to install the plugin and get it working. 
Install From WordPress Admin Panel:

1. Login to your WordPress Admin Area
2. Go to Plugins -> Add New
3. Type "Review Manager" into the Search and hit Enter.
4. Find this plugin Click "install now"
5. Activate The Plugin

=Manual Installation=

1. Download the plugin from WordPress.org repository
2. On your WordPress admin dashboard, go to Plugins -> Add New -> Upload Plugin
3. Upload the downloaded plugin file (review-manager.zip) and click Install Now
4. Activate "Review Manager" from your Plugins page.
